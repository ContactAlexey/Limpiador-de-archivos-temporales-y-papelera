# Limpiador-de-archivos-temporales-y-papelera
¿Tu disco se llena sin razón aparente? Este script de Windows limpia en segundos los archivos temporales y la papelera que el sistema acumula con el uso diario.

Este Script limpia los siguientes directorios TEMP, TMP, Prefetch y la papelera de reciclaje:

#TEMP y TMP
Aqui se crean los archivos temporales de los programas mienttras se estan ejecutando, en este caso los instaladores dejan restos, las Apps dejan  archivos provicionales y los navegadores guardan caché.

si se limpia estos directorios se liberia el espacio en el disco, se evitan los conflictos con archivos antiguos o corrusptos y alguno programas pueden arrancar mas rapido si no hay archivos basura acumulados.

#Prefetch
Guarda datos del arranque de los programas para que el sistema los abra mas rapido (mapas), recomendado en caso de que haya archivos corruptos, pero no se debe de borrar diariamente ya que podria afectar al rendimiento de los programas y del sistema porque le generas trabajo innecesario de aprendizaje de como tiene que abrir un programa de forma eficiente.

Pero en el caso de que sospechemos que haya uno de estos casos entonces si deberiamos de borrarlo:
  ·Sospechas archivos corruptos
  ·Problemas raros de arranque
  ·Apps que tardan más de lo normal sin motivo

#Papelera de reciclaje
Aqui se almacenan todos los archivos que se "Eliminan" pero aun siguen ocupando espacio, elimanr los archivos de este directorio librera espacio en el disco y evita que el sistema gestione archvios inncesarios
